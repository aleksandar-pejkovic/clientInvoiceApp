Instructions about making rest calls with JSON format examples

{
"name":"Customer name",
"address":"Street name",
"zip":"zip number",
"city":"City name",
"pib":"tax id number",
"mb":"company id",
"phone":"phone number",
"bankAccount":"bank account number",
"email":"email address"
}

GET - http://localhost:8080/customers/listAll
